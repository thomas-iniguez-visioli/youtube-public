# Project: youtube-public

## Core Value
A local YouTube-like experience for archived videos, providing a private, offline-capable, and organized way to consume content.

## Vision
To be the most reliable and user-friendly local video library for YouTube content, focusing on seamless downloads and high-quality playback.

## Current Focus
Phase: Stability and Maturity (v1.1.58+)
Goal: Transition from rapid feature development to a robust, production-ready application.

## Constraints
- Local-first architecture.
- Electron-based desktop application.
- Minimal external dependencies.
- Private usage (no external trackers).
